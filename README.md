# Poker Tracker

一个用于德州扑克牌局记录与计分的本地桌面应用。软件用于记录每场牌局的玩家、标签、初始带入、补筹、离场金额和最终盈亏，并提供基础统计与走势展示。

## 功能概览

- 牌局记录：创建牌局、选择入局玩家、记录每位玩家的补筹与离场金额。
- 自动换算：离场金额按 `离场金额 / 2000 = 离场筹码 BB` 换算，并计算每位玩家输赢。
- 玩家管理：新增玩家、修改姓名、随机头像、删除无可见牌局记录的玩家。
- 标签筛选：为牌局添加标签，并按时间、标签和关键词筛选牌局。
- 统计面板：查看指定玩家的总盈利、胜率、补筹统计、盈利走势和结果分布。
- 参数设置：配置新牌局默认初始带入 BB 和两个补筹 BB 档位。
- 桌面窗口：固定 16:9 主画面比例，使用自定义窗口标题栏。

## 本地运行

```bash
npm install
npm run dev
```

开发服务器启动后，可在浏览器中打开 Vite 输出的本地地址。

## 启动桌面应用

```bash
npm run desktop
```

该命令会先构建 Web 资源，再启动 Electron 桌面窗口。

## 打包发布

```bash
npm run desktop:pack
```

打包产物会输出到 `release/` 目录。当前配置面向 Windows，产物类型为 portable 可执行文件。

## 数据说明

- 数据保存在本机浏览器/Electron 的 `localStorage` 中。
- 当前版本不包含登录、云同步或多人共享数据。
- 清理应用数据或更换运行环境可能导致本地记录不可见，请在重要牌局前自行确认数据环境。
